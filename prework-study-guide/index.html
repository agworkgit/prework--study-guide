Prework Study Guide

HTML

  • HTML or Hyper Text Markup Language is used to give content structure
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element is a semantic element that defines a section of content
  • The h element is used for a big header, as the number goes up the text becomes smaller
  • The p element is used to write paragraphs or blocks of text
  • The ul element is used to create a bullet or unordered list and the contect is added by using a li element
  • The ol element is used for numbered or ordered lists by listing items with li as above

CSS

  • CSS or Cascading Sheet Styles is used to style the content

Git

  • Git is used to communicate with GitHub and sync files between your computer and the online servers that store your repos
  • git status: checks what branch we are currently on
  • git pull origin yourreponame: clones the server repo to your local machine
  • git push origin yourreponame: will push all the commits in your local repo to the remote server
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: used to commit all files in the working branch

JavaScript

  • JavaScript is what gives your website logic and functionality which helps the end user interact with your creation
